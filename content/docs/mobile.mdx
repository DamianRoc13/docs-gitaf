## Visión General

La aplicación móvil **GITAF** actúa como el centro de mando del sistema. Desarrollada nativamente para el ecosistema **Android**, se distribuye mediante archivo instalable (**APK**) para garantizar la compatibilidad con dispositivos sin necesidad de tienda de aplicaciones.

Su diseño prioriza dos aspectos fundamentales:
1.  **Control Total:** Gestión de los 3 nodos de audio y monitoreo de energía.
2.  **Accesibilidad Universal:** Detección automática de herramientas de asistencia (TalkBack).

## Requisitos de Instalación

Al ser una aplicación distribuida fuera de Google Play Store, requiere una configuración inicial sencilla.

### Especificaciones Técnicas
| Requisito | Detalle |
| :--- | :--- |
| **Sistema Operativo** | Android 8.0 (Oreo) o superior |
| **Formato** | Archivo `.apk` (Android Package) |
| **Conectividad** | Bluetooth 4.0+ (BLE Support) |

<Callout type="warn" title="Permisos Necesarios">
Para que el escaneo Bluetooth funcione correctamente en Android, la aplicación solicitará dos permisos críticos al iniciarse:
1.  **Ubicación (GPS):** Requerido por Android para detectar balizas BLE cercanas.
2.  **Dispositivos Cercanos:** Permiso para vincularse con hardware externo (Gitaf Pro).
</Callout>

### Cómo Instalar el APK

1.  Descarga el archivo `Gitaf_App_v1.0.apk` en tu dispositivo.
2.  Al abrirlo, Android te pedirá permiso para **"Instalar aplicaciones de fuentes desconocidas"**.
3.  Otorga el permiso al navegador o gestor de archivos.
4.  Presiona "Instalar" y luego "Abrir".
