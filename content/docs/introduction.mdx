## ¿Qué es GITAF?

**GITAF** (Guía Interactiva de Tiro para Atletas con Discapacidad Visual en Fútbol) es un sistema de orientación audible inalámbrico diseñado para el entrenamiento de fútbol adaptado. Este proyecto proporciona autonomía a deportistas con discapacidad visual durante la práctica del Fútbol 5.

## Características Principales

- **Orientación por Audio**: Sistema de posicionamiento mediante señales sonoras
-**Control Móvil**: Aplicación Android accesible con soporte TalkBack
-**Inalámbrico**: Comunicación ESP-NOW sin necesidad de cables
-**Accesible**: Diseñado específicamente para personas con discapacidad visual
-**Eficiente**: Gestión inteligente de energía para larga duración

## Objetivo del Proyecto

El sistema GITAF fue desarrollado para **ANVIZSUC** (Asociación Nacional de Videntes al Servicio de las Personas Ciegas del Zulia), con el objetivo de mejorar la autonomía en el tiro y la orientación espacial de los deportistas sin depender de un guía humano durante los entrenamientos.

## Tecnologías Utilizadas

- **ESP32**: Microcontrolador con WiFi y Bluetooth integrado
- **ESP-NOW**: Protocolo de comunicación inalámbrica de bajo consumo
- **Android**: Aplicación móvil nativa con accesibilidad
- **Bluetooth LE**: Conexión de bajo consumo entre app y hardware
