## Sistema de Orientación Audible para Fútbol Adaptado

**GITAF PRO** es una solución tecnológica integral diseñada para brindar autonomía a los atletas con discapacidad visual. Mediante el uso de una red de nodos inteligentes basados en **ESP32**, el sistema emite señales acústicas precisas que permiten al deportista orientarse en el campo de juego y localizar las porterías sin intervención humana constante.

## Ruta de Implementación Rápida

1. **[Identificación](/docs/firmware/setup)**: Obtén las direcciones MAC de tus 3 placas ESP32.
2. **[Ensamblaje](/docs/hardware/diagrams)**: Arma los circuitos utilizando el filtro anti-ruido de 1000uF.
3. **[Firmware](/docs/firmware/communication)**: Carga el código correspondiente al Maestro y a los Esclavos.
4. **[Preparación de Audio](/docs/firmware/audio)**: Configura la MicroSD con la carpeta `mp3` y los archivos `0001.mp3`.
5. **[Control](/docs/mobile)**: Instala el APK en Android y conéctate al dispositivo "Gitaf Pro".

## Tecnologías Implementadas

- **ESP32 WROOM**: Microcontrolador dual-core para procesamiento simultáneo de red y audio.
- **ESP-NOW**: Protocolo propietario de baja latencia para comunicación entre nodos en campo.
- **BLE (Bluetooth Low Energy)**: Enlace de telemetría y mando con dispositivos móviles.
- **INA219**: Medición digital de alta precisión para protección de baterías LiPo.
- **PAM8403 + DFPlayer**: Sistema híbrido para reproducción de MP3 de alta potencia (5W).
